FAQ

Where is the Webplayer / Download list?

Right now, these have to be inserted manually via so calles shortcodes. They are [podlove-web-player] and [podlove-episode-downloads].

There are compatibility issues with the "Jetpack" plugin. If you use it, you might need to turn it off.

Where are my Feeds?

Go to Podlove > Shows and look for the "Feeds" column. For everything to work correctly you have to set a show slug and a feed slug! The feed URL will then look like this: http://www.mypodcast.fm/feed/<show-slug>/<feed-slug>

How do I add Flattr Integration to my episodes?

  1. If you haven't already, get the official Flattr plugin here: http://wordpress.org/extend/plugins/flattr/
  2. Find the setting Flattr > Advanced Settings > Flattrable content > Post Types and check podcast. Save changes.
  3. There is no step 3 ;)
